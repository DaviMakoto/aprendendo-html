Perguntas frequentes

  1. O que fazer se não quero me exercitar?

    2. Procure outra coisa para fazer, como por exemplo: assistir vídeos, séries, entre outros.

  2. Como faço para logar minha conta?

    3. Primeiro deve-se criar sua conta, depois apenas colocar os mesmo dados que usou para se cadastrar.

  3. Como faço para ver um vídeo? Sou completamente amadora com tecnologias atuais.

    4. Primeiro, deve-se apertar lá na barra de pesquisa, depois escrever "Youtube" clique no resultado que aparecer e aproveite vendo vídeos do seu interesse.

  4. Como faço para ver o seu perfil do instagram?

    5. Simples! Você inclusive pode apenas clicar para acessar a aba "contato" que lá está o meu perfil.
